# Web interface

This overview provides information on how to use Omnom's web interface and features.

## Dashboard

Your personal dashboard provides an overview of your activity:

- **Statistics**: View bookmark counts for the current week, month, and year
- **Frequent Tags**: Quick access to your most-used tags with usage counts
- **Latest Bookmarks**: Recent bookmarks you've saved

Access the dashboard with the "Home" navigation link after logging in.

## Bookmarks

### Creating Bookmarks

Create a new bookmark by filling out the following fields:

- **URL** (required): The web address you want to save
- **Title**: Custom title for the bookmark (optional)
- **Tags**: Comma-separated tags for organization
- **Collection**: Assign to a specific collection (optional)
- **Notes**: Personal notes about the bookmark
- **Public**: Check to make the bookmark visible to other users
- **Unread**: Mark as unread for later review

### Viewing Bookmarks

**My Bookmarks** - Personal bookmark collection of private and public bookmarks

**Public Bookmarks** - Bookmarks shared by all users (with RSS feed available)

### Editing Bookmarks

From the bookmark edit page, you can:

- Modify the title, notes, and collection
- Toggle public/private visibility
- Mark as read/unread
- **Manage Tags**: Add new tags or remove existing ones
- **Manage Snapshots**: View or delete saved snapshots
- **Delete**: Permanently remove the bookmark

### Bookmark Search and Filtering

Use the advanced search options to filter bookmarks by:

- **Text Search**: Search in titles, URLs, and content
- **Domain**: Filter by website domain
- **Owner**: Filter by username (for public bookmarks)
- **Tags**: Filter by specific tags
- **Date Range**: Filter by date created
- **Sort By**: Sort results by date, title, or relevance

**Frequent Tags Panel**: Quick filter by clicking on popular tags showing usage counts.

## Snapshots

Omnom captures website snapshots as they appear in your browser, preserving dynamic content.

### Snapshot Features

- **Multiple Snapshots**: Save multiple versions of the same URL over time
- **Resource Summary**: View the size and details of saved snapshots
- **Compare/Diff Views**: Compare different versions to see what changed

### Finding Snapshots

Use the **Snapshot Search** feature to:

1. Search by domain or URL
2. View all snapshots matching your query
3. Access snapshot details including:
   - Original URL
   - Snapshot date and time
   - File size

### Snapshot Diff

Compare two snapshots of the same or different URLs:

- **Side-by-Side View**: See differences in parallel columns
- **Unified Diff**: Traditional diff format
- View detailed changes in content and structure

## Feeds

Aggregate and read RSS/Atom feeds and ActivityPub content.

### Managing Feeds

**Add a Feed**: Subscribe to RSS/Atom feeds or ActivityPub accounts

**Edit Feed**:
- Change the feed name
- Delete the feed

### Reading Feeds

**Feed Items Display**:
- Feed name and source
- Title and content preview
- Publication date
- Link to original source in case of repost
- Original author in case of repost

**Archive Items**: Archive items individually using the "Archive" button on the top right of each item or mark the current page of items as read with the "Archive Page" button

### Feed Search

Search within your subscribed feeds:

- Search across all feeds or within a specific feed
- View item count for search results
- Supports both feed items and bookmarked items

## Collections

Organize bookmarks into hierarchical collections (folders).

### Creating Collections

- **Name**: Choose a descriptive name
- **Parent**: Optionally nest within another collection

### Editing Collections

- Rename collections
- Change parent collection (move to different location)

### Using Collections

- Assign bookmarks to collections when creating or editing
- Filter bookmarks by collection
- Create nested structures for better organization

## Search

### Global Search

Search across all your content and view it as a combined result set including

- Feed items
- Bookmarks

### Feed-Specific Search

Search within a specific feed or across all feeds with:

- Feed name displayed in results
- Total item count
- Mixed results from feeds and bookmarks

### Search Tips

- Use the text filter for quick searches
- Combine with advanced filters for precise results
- Results show the most relevant matches first

## Profile Management

### Account Information

- View username and email
- Check storage usage (size of all your snapshots)
- Logout button

### Addon Tokens

Manage browser extension tokens for the Omnom addon:

**View Tokens**: Click "Show Addon Tokens" to display existing tokens

**Generate Token**: Create new addon tokens for browser extensions

**Delete Token**: Remove tokens you no longer use

Addon tokens allow the browser extension to communicate with your Omnom instance securely.

## RSS Feeds

Omnom provides RSS feeds for:

- **Public Bookmarks**: Subscribe to all public bookmarks
- **Search Results**: RSS feed of search results

Look for the RSS icon in the top-right corner of these pages.

## Tips and Best Practices

1. **Use Tags Liberally**: Tags make finding bookmarks easier later
2. **Public vs Private**: Keep sensitive bookmarks private
3. **Collections**: Use collections to organize by project or topic
4. **Unread Flag**: Mark items as unread to create a reading list
5. **Snapshots**: Enable snapshots to preserve content even if the original site changes or disappears
6. **Regular Archiving**: Archive read feed items to keep your unread list manageable
7. **Search Filters**: Combine multiple filters for precise searches
