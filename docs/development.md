# Development

Omnom is an evolving project. Contributions from the community are highly valued. Whether you're a developer, a writer, a tester, or simply an enthusiastic user, there are many ways you can help shape the future of Omnom.

### Code Contributions

If you're a developer, consider contributing code! This can include fixing existing bugs, implementing new features, or refactoring code to improve performance and maintainability. Check out the [project repository](https://github.com/asciimoo/omnom) for contribution guidelines and current development tasks.

### Bug Reporting

Encounter a bug? Please [report it](https://github.com/asciimoo/omnom/issues)! Detailed bug reports are crucial for identifying and fixing issues. When reporting a bug, include steps to reproduce it, your environment (e.g., operating system, browser, Omnom version), and any relevant error messages or screenshots.

### Feature Requests and Ideas

Have an idea for a new feature or a suggestion to improve an existing one? We'd love to hear it! Please share your thoughts through the project's [issue tracker](https://github.com/asciimoo/omnom/issues). Clear descriptions of the problem your feature would solve and how it might work are always helpful.

### Documentation Improvement

Clear and comprehensive documentation is vital. If you find any part of the documentation unclear, outdated, or missing, your contributions are welcome. This includes fixing typos, rephrasing sections, or even writing new guides and tutorials.

### Testing and Feedback

Help us ensure Omnom is stable and user-friendly by testing newest release versions or specific features. Your feedback on usability, performance, and any issues you discover is invaluable.

### Translation

To make Omnom accessible to users worldwide, we need your help with [translations](https://hosted.weblate.org/projects/omnom/)! If Omnom is not yet available in your preferred language, or if you'd like to improve existing translations.

### Community Support

Assist other users by answering questions and sharing your expertise in community forums, chat channels. Helping others use Omnom effectively fosters a stronger community.
